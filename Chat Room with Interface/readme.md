# Chat Room Projekt

## Überblick
Dieses Projekt ist eine Chat-Anwendung, die mit LabVIEW entwickelt wurde. Sie ermöglicht es Benutzern, Nachrichten in einem Chat-Fenster auszutauschen.

## Projektstruktur
- **Launcher.vi**: Startet die Anwendung
- **Server.lvlib**: Enthält die Server-Funktionalität
- **Chat Window Model.lvlib**: Verwaltet die Chat-Daten
- **Chat Window UI Controller.lvlib**: Steuert die Benutzeroberfläche
- **Chat Window Interface.lvlib**: Schnittstelle für die Chat-Kommunikation
- **Controls**: Enthält benutzerdefinierte Steuerelemente
- **Documentation**: Enthält die Dokumentations-VI

## Systemvoraussetzungen
- LabVIEW 2020 oder neuer
- Windows-Betriebssystem

## Installation
1. Öffnen Sie die Projektdatei "Chat Room.lvproj" in LabVIEW
2. Die benötigten Abhängigkeiten werden automatisch geladen

## Ausführen der Anwendung
1. Öffnen Sie "Launcher.vi" im Projekt
2. Führen Sie die VI aus, um die Chat-Anwendung zu starten

## Erstellen einer ausführbaren Datei
Das Projekt enthält eine Build-Spezifikation "EXE". Diese erstellt eine ausführbare Datei "ChatWindow.exe" im Ordner "builds".
